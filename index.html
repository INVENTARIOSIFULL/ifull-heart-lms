Deploy en GitHub Pages
1. Sube el archivo HTML a tu repo (ej. docs/index.html)
2. Settings → Pages → Source: /docs
3. El LMS queda en https://TU_USUARIO.github.io/ifull-heart-lms/
Para producción, usa variables de entorno con Vite/Next.js en lugar de pegar la API key directamente. Configura Firestore Security Rules para restringir acceso por UID.
